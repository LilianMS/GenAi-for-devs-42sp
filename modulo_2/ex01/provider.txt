1. Passo a passo para obter uma chave de API para o Google AI Studio

    1.1 - Acessar o Google AI Studio:
        Vá para o Google AI Studio e faça login com sua conta Google.

    1.2 - Obter chave de API:
        No painel esquerdo ou na parte superior da interface, clique no botão "Obter chave de API".
    
    1.3 - Criar chave de API: Em seguida, clique em "Criar chave de API".
    
    1.4 - Selecionar projeto (ou criar novo): Escolha um projeto existente do Google Cloud ou opte por criar uma chave de API em um novo projeto.
    
    1.5 - Copiar a chave de API: Uma vez gerada, copie a chave de API apresentada. Esta é a string de caracteres que você usará para autenticar suas solicitações à API Gemini. 
    
    Observação:
        É crucial manter sua chave de API segura.
        Não a incorpore diretamente em seu código e evite compartilhá-la.
        É recomendado usar variáveis de ambiente para armazenar chaves de API. 


2. Modelos de LLM suportados e seus limites de taxa

    1. Gemini 1.5 Flash (gemini-1.5-flash)
        Entrada: Texto, imagem, áudio, vídeo.
        Saída: Texto, código, JSON.
        Otimizado para: Eficiência de custos, alta taxa de transferência, baixa latência.
        Contexto de Janela: 1 milhão de tokens.
        Limites de Taxa (padrão): 60 RPM, 1.000.000 TPM.

    2. Gemini 1.5 Pro (gemini-1.5-pro)
        Entrada: Texto, imagem, áudio, vídeo.
        Saída: Texto, código, JSON.
        Otimizado para: Tarefas complexas, raciocínio avançado, compreensão multimodal.
        Contexto de Janela: 1 milhão de tokens.
        Limites de Taxa (padrão): 60 RPM, 1.000.000 TPM.

    3. Gemini 1.0 Pro (gemini-1.0-pro)
        Entrada: Texto.
        Saída: Texto, código, JSON.
        Otimizado para: Raciocínio, codificação e geração de texto de alta qualidade.
        Contexto de Janela: 32.000 tokens.
        Limites de Taxa (padrão): 60 RPM, 1.000.000 TPM.

    4. Gemini 1.0 Pro Vision (gemini-1.0-pro-vision)
        Entrada: Imagens e texto.
        Saída: Texto.
        Otimizado para: Compreensão de imagens e documentos.
        Contexto de Janela: 16.000 tokens.
        Limites de Taxa (padrão): 60 RPM, 1.000.000 TPM.
        
    Observação:
        Os limites de taxa podem variar dependendo do nível de acesso (gratuito ou pago) e das especificidades de cada modelo.
        O nível gratuito oferece 60 solicitações por minuto, enquanto os níveis pagos proporcionam maior capacidade e privacidade de dados.
